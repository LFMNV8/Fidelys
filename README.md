# Fidelys

Fidelys é um site de consultoria financeira desenvolvido para ajudar pessoas e empresas a organizar suas finanças e atingir suas metas.  

## Tecnologias Usadas

- **Front-end**:
  - HTML5
  - CSS3
  - Bootstrap
  - JavaScript
- **Back-end**:
  - Python
  - Django
- **Banco de Dados**:
  - MySQL
